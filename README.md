This repository provides R and Python implementations for LiDAR point cloud processing and machine learning-based forest Aboveground Biomass (AGB) estimation. The workflow includes:

1. LiDAR Processing: Generating Canopy Height Models (CHM) and applying watershed algorithms for individual tree segmentation.

2. AGB Mapping: Integrating LiDAR-derived structural features (elevation and intensity) with XGBoost to map biomass at scale.
